# Avaliação N1 - Aplicação DevOps

Este projeto consiste numa aplicação web simples desenvolvida com **Node.js** e o framework **Express.js**. O objetivo principal da atividade é demonstrar o domínio do fluxo de trabalho em DevOps utilizando Git e GitHub, especificamente a gestão de branches e a criação de endpoints.

## 👥 Integrantes e Responsabilidades
- **João**: Responsável pela Gestão de Configuração e DevOps. Realizou a estruturação do repositório, criação e gerenciamento de todas as *feature branches*, resolução de conflitos, configuração do `.gitignore` e deploy das rotas no GitHub.
- **Lorenzo**: Responsável pelo Desenvolvimento Backend. Atuou na implementação da lógica do servidor Express, configuração das rotas de API e estruturação do objeto JSON de integrantes.
- **Itallo**: Responsável pelo Frontend e Documentação. Realizou a criação da página `index.html`, estruturação das pastas de visualização (`views`) e elaboração da documentação técnica (README).

## 🛠️ Tecnologias Utilizadas
- **Node.js**: Ambiente de execução.
- **Express.js**: Framework para construção das rotas.
- **Git/GitHub**: Controlo de versões e organização em branches.
